# markdown-it-example

## Overview

markdown-it で各種タグの動作をカスタマイズしたサンプルです。
This is an example of customizing the behavior of various tags with markdown-it.

## Requirement

- Node.js (v16.14.2)

## Setup

```shell
# 依存関係のパッケージをインストール
$ npm i

# 開発環境で実行 (http://localhost:3000/)
$ npm run dev

# 本番環境向けにビルド
$ npm run build
```
